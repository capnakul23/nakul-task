# TaskFlow — Team Task Manager

A full-stack project & task management web app with role-based access
control, Kanban workflows, and real-time dashboards — built for teams to
collaborate efficiently.

Live Demo: https://nakul-task-gshm.vercel.app/
GitHub Repo:https://github.com/capnakul23/nakul-task

------------------------------------------------------------------------

🚀 Key Highlights

-   Secure JWT Authentication (Access + Refresh tokens)
-   Full Project Lifecycle Management
-   Advanced Task System with Kanban Board
-   Real-Time Dashboard & Analytics
-   Team & Role Management (Admin / Member / Viewer)
-   Optimized API with Rate Limiting & Validation
-   Modern UI with smooth animations

------------------------------------------------------------------------

📁 Features

Authentication: - JWT-based login & signup - Auto token refresh -
Password hashing with bcrypt - Profile & avatar management

Project Management: - Create, update, delete projects - Assign roles
(Admin / Member / Viewer) - Progress tracking - Archive projects

Task Management: - Create & assign tasks - Kanban workflow (To Do →
Done) - Comments & estimated hours - Overdue alerts

Dashboard: - Task stats & analytics - Activity feed - Project insights

Team Management: - Manage users (Admin only) - Role switching -
Activate/deactivate users

------------------------------------------------------------------------

🏗️ Tech Stack

Backend: - Node.js + Express - MongoDB + Mongoose - JWT, bcrypt,
express-validator - Helmet, CORS, Rate Limiting

Frontend: - React 18 + Vite - React Router - TanStack Query - Zustand -
Tailwind CSS - Axios, Framer Motion

------------------------------------------------------------------------

⚙️ Setup (Quick)

1.  Clone Repo: git clone https://github.com/capnakul23/nakul-task.git

2.  Backend: cd backend npm install npm run dev

3.  Frontend: cd frontend npm install npm run dev
